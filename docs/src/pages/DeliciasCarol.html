Delícias da Carol

Esta página apresenta o Delícias da Carol, sistema completo para gestão comercial que desenvolvi, com controle integrado de vendas, estoque e finanças, incluindo visão geral do projeto e tecnologias utilizadas.

Overview

O Delícias da Carol é um projeto que desenvolvi com o objetivo de facilitar o controle de vendas, estoque e finanças para pequenos e médios negócios. Esta solução fullstack foi criada para oferecer uma experiência completa e eficiente na gestão comercial. Principais funcionalidades que implementei:

  • Cadastro de produtos: Desenvolvi um sistema completo para registro de produtos com controle detalhado de estoque
  • Registro de vendas: Criei um fluxo otimizado para registrar transações comerciais com agilidade e precisão
  • Controle financeiro: Implementei um módulo completo para gerenciamento de pagamentos e fluxo de caixa
  • Gráficos de acompanhamento: Desenvolvi visualizações gráficas para monitoramento do desempenho comercial

Destaques do projeto que construí:

  • Interface altamente otimizada para agilizar o processo de vendas
  • Sistema de alertas para estoque baixo que implementei
  • Relatórios completos que desenvolvi para vendas, estoque e financeiro
  • Solução 100% autoral, criada para atender necessidades reais do mercado

Projetei o Delícias da Carol para ser uma ferramenta prática, robusta e eficiente, que realmente facilita o dia a dia do usuário.

Arquitetura e Tecnologias Utilizadas

Desenvolvi o Delícias da Carol em duas camadas principais: back-end e front-end, com atenção especial à performance e usabilidade.

  • Back-end: Arquitetura MSC que construí utilizando Node.js com TypeScript. Implementei autenticação com JWT e proteção de senhas com Bcrypt. Para o banco de dados, utilizei MySQL com Prisma como ORM - escolha que otimizou significativamente o desenvolvimento.
  • Front-end Mobile: Desenvolvi o aplicativo em React Native com Expo e TypeScript, criando uma solução multiplataforma com performance nativa. Implementei gráficos interativos para melhor visualização dos dados.
  • Sistema de estoque: Criei um controle em tempo real com atualizações instantâneas e alertas personalizáveis
  • Módulo financeiro: Desenvolvi um sistema completo para acompanhamento de recebíveis e fluxo de caixa
  • Exportação de dados: Implementei funcionalidade para gerar relatórios em diversos formatos

Esta arquitetura que construí sozinho resulta em uma plataforma robusta e eficiente, demonstrando minha capacidade de desenvolver soluções completas.
